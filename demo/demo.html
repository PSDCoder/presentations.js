Presentations.js

Simple presentations jQuery plugin.

  • Presentations.js

    Presentation.js is simple but powerful presentation jQuery plugin.

    Main features of plugin:

    • You can add many presentations to page
    • Rich configuration object
    • Simple API
    • FullScreen mode (Supports only modern browsers. More)
    • Keyboards hotkeys for:
      • Fullscreen mode (f)
      • Flipped slides (← & →)

  • Basic usage:

    1. If you want use pagination functionality - include on page js library file lib/paginator.js
    2. Include on page main js library file lib/presentations.js
    3. Include basic styles for presentations lib/presentations.css
    4. Also you can include beautiful styles for presentations from this demo demo/css/styles.css
    5. Create basic markup for presentation: 
      <div class="presentation">
    <ul class="presentation-slides">
        <li class="presentation-slides__slide">
            First slide content
        </li>
        <li class="presentation-slides__slide">
            Second slide content
        </li>
    </ul>
</div>
    6. Initialize presentations: 
      (function ($, undefined) {
    $(function () {
        $('.presentation').presentations({});
    });
})(jQuery);
    7. Now you have pretty presentation. You can manage it with $('.presentation').presentations('method'[,'param']);

  • Plugin API:

    Method Description Call example
    init Initialize plugin. For configure presentation you can pass object with options. $('.presentation').presentations({});
    $('.presentation').presentations('init', {});
    destroy This method will be destroy presentation to state before initialization. $('.presentation').presentations('destroy');
    next This method changes the slide to the next. $('.presentation').presentations('next');
    previous This method changes the slide to the previous. $('.presentation').presentations('previous');
    setActive This method sets N slide as active. $('.presentation').presentations('setActive', 10);

  • Plugin options:

    Option Default value Description
    activeSlide 1 Sets N slide as active after initialization plugin.
    slideWithKeyboard true Enables switching slides with keyboard (With ← & →).
    loopMode false Enables looped mode for switching slides.
    pagination true Enables pagination for presentation. See paginationOptions option for setting up pagination.
    paginationOptions 
    {
    afterFirst: 1,
    beforeLast: 1,
    beforeCurrent: 1,
    afterCurrent: 1,
    prevText: 'Prev',
    nextText: 'Next'
}
    		For more information see Paginator.js plugin documentation
    progressBar true Enables progress bar for presentation.
    arrows true Enables arrows buttons for switching slides.
    fullScreen true Enables fullscreen mode for presentation. This feature will be available for user if his browser supports HTML5 fullscreen API.
    fullScreenKey 'f' Key for switching to fullscreen mode and back.
    containerClass 'presentation' Plugin will be find presentation container in DOM by this CSS class.
    fullScreenClass 'fullscreen-mode' This CSS class will be added to body DOM element when presentation will be switched to fullscreen mode.
    nextButtonClass 'presentation__button_action_next' This CSS class will be added to generated 'Next' button
    nextButtonText 'Next' This text or html will have a generated 'Next' button
    previousButtonClass 'presentation__button_action_previous' This CSS class will be added to generated 'Previous' button
    previousButtonText 'Previous' This text or html will have a generated 'Previous' button
    slidesListClass 'presentation-slides' Plugin will be find container of slides in DOM by this CSS class.
    slideClass 'presentation-slides__slide' Plugin will be find every slide in slide's container by this CSS class.
    activeSlideClass 'presentation-slides__slide_state_active' Plugin will add this CSS class to current visible slide.
    paginationClass 'presentation-navigation' This CSS class will have generated pagination.
    paginationItemClass 'presentation-navigation__item' This CSS class will have every item of generated pagination.
    progressBarClass 'presentation-progress-bar' This CSS class will have generated progress bar container.
    progressBarCountClass 'presentation-progress-bar__count' This CSS class will have generated bar in him container.
    fullScreenButtonClass 'presentation__button_action_fullscreen' This CSS class will be added to generated 'Fullscreen' button
    fullScreenButtonText 'Fullscreen (f)' This text or html will have a generated 'Fullscreen' button in mormal mode
    fullScreenButtonTextEnabled 'Exit from fullscreen (f)' This text or html will have a generated 'Fullscreen' button in fullscreen mode

  • Plugin Events:

    Event Callback config Description
    initEnd function ($presentation){} This event will be fired after initialization presentation.
    destroyEnd function ($presentation){} This event will be fired after destroying presentation.
    changeSlide function ($presentation, showIndex){} This event will be fired when slide was changed.
    nextSlide function ($presentation, showIndex){} This event will be fired when slide was changed to the next.
    previousSlide function ($presentation, showIndex){} This event will be fired when slide was changed to the previous.
    firstSlide function ($presentation, showIndex){} This event will be fired when was selected first slide.
    lastSlide function ($presentation, showIndex){} This event will be fired when was selected last slide.
    fullScreen function ($presentation){} This event will be fired when presentation was switched to fullscreen mode.
    fullScreenExit function ($presentation){} This event will be fired when presentation was switched to normal mode from fullscreen.
